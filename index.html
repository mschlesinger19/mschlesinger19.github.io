My Projects

Magic Mirror

Raspberry Pi

Using facial recognition and open source code found on GitHub, I created a MagicMirror product to display commute times, weather, phone notifications, to-do lists, and sports scores personalized for the person standing in front of the household mirror.

FlipIt

Android Application

I created an Android application for realtors to assist them in recording estimates for rental and flip properties as they do their "initial walk-throughs". With a local database, users can easily save and edit previous entries if they have any changes. Users also have the ability to email their "spreadsheet" and use any email application to send a simple text formatted message with all of the values on the entry screen.

Not released in Google Play

Hold My Beer

Windows Appliciation

A few classmates and myself created a Window's phone application that won Microsoft's Grand Prize at the University of Miami's 24-hour Hackathon. This application is like Untapped (which was not popular at the time) where you can search a beer database (API found online) to view the beer's description, ABV, location, average user rating, and even save your own rating and notes about the beer for your own reference. We attempted to implement label recognition as a search cirteria but it was not 100% successful due to the time.

Not released in Windows Store
